🌙 Sonhei — Discord Bot

Um bot de Discord moderno, leve e personalizável, criado para melhorar a experiência do seu servidor com automações, utilidades e interações inteligentes.

🚀 Tecnologias utilizadas
JavaScript (Node.js)
discord.js
📦 Funcionalidades
🎫 Sistema de tickets
📊 Sistema de XP por mensagem
🔨 Moderação (ban, mute, etc.)
🧩 Painéis interativos (botões e menus)
🖼️ Sistema de banners/customizações
🔔 Notificações automáticas (ex: novos vídeos)
👥 Marcação automática de staff
